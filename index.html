Quick Link Highlighter Plugin

A lightweight WordPress plugin to highlight external links in posts, pages, and custom post types.

Features

  • Automatically detects external links.
  • Adds a visual indicator (icon or color change) to external links.
  • Lightweight and easy to use.

Installation

  1. Download the plugin from GitHub.
  2. Upload the quick-link-highlighter folder to your WordPress site's /wp-content/plugins/ directory.
  3. Activate the plugin in the WordPress admin dashboard under Plugins.
