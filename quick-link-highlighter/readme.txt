A lightweight plugin to highlight external links in posts, pages, and custom post types.

== Description ==
Quick Link Highlighter automatically detects external links in your content and adds a visual indicator (like an icon or color change) to make them stand out.

== Installation ==
1. Upload the `quick-link-highlighter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
